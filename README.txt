onion_harvester.py
==================

Script to harvest TOR hidden services' onion addresses.



Usage
=====

$ onion_harvester.py [options]


Options:

-h 	 	 	 Show this help

-i 	 <inputfile> 	 Read URLs from inputfile

-o 	 <outputfile> 	 Write harvested onions to outputfile (default: harvest.txt)

-s 	 <searchterm> 	 Harvest Google search results (default: 'onion links')



Default
=======

If no option / argument is passed the Onion Harvester just crops what it can find on Google's field.
